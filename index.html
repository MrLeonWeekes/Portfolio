    Mongo DB

    MongoDB works in a document-oriented NoSQL form. It is an open-source Database Management System (DBMS). MongoDB is considered to be a NoSQL database since it doesn’t depend on the relational model. MongoDB, on the other hand, stores its data records in structures known as documents. Mongo allows you to group multiple documents into a structure known as a collection, which can be further grouped into separate databases.
